  def hello():

And this is a C code block:

#include <stdio.h>
int main()
{
    printf("Hello, World!");
    return 0;
}

def multiply(a: float, b: float) -> float:
    """
    Multiply two numbers.

    :param a: First number.
    :param b: Second number.
    :return: The product of a and b.
    """
    return a * b

This creates an equation:

\[a^2 + b^2 = c^2\]
